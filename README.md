## Описание
Приложение мессенджера, написанное на ts + ejs.compile + scss, собирается в parcel, запускается на express, задеплоено в [netlify](https://storied-sprite-3c3831.netlify.app/)

## Команды для запуска
`yarn` - установка зависимостей
`yarn serve` - запуск приложения в режиме разработки
`yarn build` - запустить сборку статики
`yarn start` - запустить express-сервер для раздачи статики
